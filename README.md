# variaciones-meteorologicas-oceanicas-cantabria
Análisis estadístico de la variabilidad meteorológica(Tama: 1972-2025; Santander: 1961-2025) y oceánica (Santander: 1943-2017) en  Cantabria: extremos térmicos, régimen de precipitación y tendencias del  nivel del mar. Pipeline reproducible en R con tests no  paramétricos. TFM Máster en Ciencia de Datos — Universidad de Cantabria.
